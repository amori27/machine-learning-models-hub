# Security Policy

## Supported Versions

We currently support the latest published version with security updates.

| Version | Supported |
|---------|-----------|
| latest  | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability in **Machine Learning Models Hub**, please report it by opening an issue at:

https://github.com/amori27/machine-learning-models-hub/issues

Do **not** disclose the vulnerability publicly until it has been addressed. We will respond within 7 business days and work to resolve the issue as quickly as possible.
